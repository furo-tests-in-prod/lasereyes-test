Setup Instructions

  1. Install the Unisat wallet browser extension from https://unisat.io/
  2. Set up a React development environment with webpack or vite
  3. Install the LaserEyes SDK: npm install @omnisat/lasereyes
  4. Use the provided React components to interact with Bitcoin wallets

Note: This is a demo environment. To fully use the LaserEyes SDK, you need to build and run this application with a proper bundler.
